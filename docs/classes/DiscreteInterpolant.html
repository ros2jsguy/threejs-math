Class DiscreteInterpolant

Interpolant that evaluates to the sample value at the position preceding the parameter.

Examples

const interpolant = new DiscreteInterpolant(
new Float32Array( 2 ),
new Float32Array( 2 ),
1,
new Float32Array( 1 )
);

interpolant.evaluate( 0.5 );
